JOC DEL PINGÜ - PINGU_G09-IAN.RUBIO-OSCAR.MUÑOZ-GABRIEL.MILEA

Descripción
Juego de mesa por consola desarrollado en Java con arquitectura MVC.
El objetivo es llegar a la casilla 50 mientras se evitan osos, agujeros de hielo y se interactúa con focas amigables.

Características
- Tablero de 50 casillas aleatorias
- Jugadores humanos e IA con diferentes comportamientos
- Inventario con límites (peces, bolas de nieve, dados)
- Eventos aleatorios
- Interacciones especiales (osos, focas, trineos)
- Sistema de turnos
- Persistencia preparada para base de datos

Tecnologías
- Java 17
- Eclipse IDE
- MySQL (preparado para futuro)
- Maven (gestión de dependencias)
